# JavaScript DSA Practice

This project is a personal learning workspace for practicing JavaScript and building DSA fundamentals step by step.

## Overview

The goal of this repository is to help you practice:

- JavaScript basics
- loops
- arrays
- problem solving
- DSA thinking
- time complexity basics

## Project Structure

`loops/`
Contains beginner loop practice and examples for DSA-style learning.

`loops/loops.js`
Includes loop notes, examples, and small practice programs.

`loops/index.html`
Loads JavaScript in the browser so you can test loop code using `prompt()` and `console.log()`.

`loops/txt file`
Can be used for notes or rough practice.

## Current Focus

Right now the main topic is **loops in JavaScript**.

This includes:

- `for` loops
- `while` loops
- increment and decrement logic
- summation problems
- input validation
- basic time complexity ideas like `O(n)` and `O(n^2)`

## How To Use

Open the relevant practice folder and run or test the files there.

For the loops topic:

- open `loops/index.html` in a browser
- run the JavaScript and enter values when prompted
- edit `loops/loops.js` to practice different loop questions

## Learning Approach

A good way to study in this repo is:

- read the example
- predict the output
- run the code
- change one thing at a time
- solve the same problem on your own
